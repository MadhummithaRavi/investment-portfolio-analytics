# Daily Investment Portfolio Analytics Pipeline

## Overview
A personal project built to gain hands-on experience with the modern data stack using BigQuery, dbt, Airflow, and GitHub.

## Architecture
CSV → BigQuery → dbt → Portfolio Analytics → Airflow

## Tools Used
- GCP BigQuery
- dbt Core
- Apache Airflow
- Git
- GitHub

## Outputs
- Portfolio Value by Stock
- Automated dbt Transformations
- Data Quality Tests

## Learning Outcomes
- Built dbt models using ref()
- Implemented dbt tests
- Created Airflow DAGs
- Used Git and GitHub for version control
